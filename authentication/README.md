# Flask Auth example

In this examplewe create a simple authentication and authorization mechanism:
- Three roles: employee, manager, admin 
- One dashboard per role
- Passwords hashed

## How to run

1. Create a virtual environment: ```python -m venv venv```
1. Activate (Windows: ```venv\Scripts\Activate```) 
1. Install flask: ```pip install flask```
1. Run: ```python app.py```
